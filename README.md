# 📦 Fly Emirates - Система регистрации пользователей

## 🎯 Проект готов к использованию!

Реализована полнофункциональная система регистрации и входа пользователей с интеграцией Google Sheets.

## 📂 Созданные файлы

- ✅ `register.html` - Страница регистрации с полями: Имя, Email, Возраст
- ✅ `login.html` - Страница входа по Email

## ⚙️ Настройка (кратко)

1. **Создайте Google Таблицу**
   - Добавьте заголовки в первую строку

2. **Настройте Apps Script**
   - Расширения → Apps Script
   - Вставьте код из `google-apps-script.js`
   - Разверните как веб-приложение (доступ: "Все")

3. **Обновите HTML**
   - В `register.html` и `login.html` замените `YOUR_GOOGLE_APPS_SCRIPT_URL_HERE`
   - На ваш URL из Apps Script

4. **Готово!**
   - Откройте `register.html` в браузере
   - Зарегистрируйте тестового пользователя
   - Проверьте Google Таблицу

## 🛠️ Возможные проблемы и решения

### ❌ "Ошибка при регистрации"
**Решение:**
- Проверьте URL в HTML файлах
- Убедитесь, что скрипт развернут как веб-приложение
- Проверьте настройку "У кого есть доступ: Все"

### ❌ "Данные не сохраняются"
**Решение:**
- Проверьте заголовки в таблице (первая строка)
- Запустите функцию `testSetup` в Apps Script
- Проверьте права доступа к таблице

### ❌ "Access denied"
**Решение:**
- Повторите авторизацию скрипта
- Убедитесь, что выбрано "Выполнять от имени: Мой аккаунт"
